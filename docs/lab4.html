เกี่ยวกับโปรเจกต์

Lab04 เป็นการพัฒนาแอปพลิเคชันแบบไฮบริด (Hybrid Mobile App) สำหรับจัดการและดูรูปภาพ โดยใช้ Vue.js, Ionic Framework และ Capacitor เพื่อสร้างแอปพลิเคชันที่สามารถทำงานได้ทั้งบนเว็บและมือถือ

ฟีเจอร์หลัก
  • ถ่ายรูปและบันทึกลงในอุปกรณ์
  • เลือกรูปภาพจากแกลเลอรี่
  • แสดงรูปภาพในรูปแบบกริด
  • ลบและจัดการรูปภาพ
  • ตอบสนองต่อขนาดหน้าจอต่างๆ (Responsive)
  • ออกแบบโดย Ionic Components

หน้าจอตัวอย่าง

หน้าแรก (Home Screen)
แกลเลอรี่รูปภาพ (Gallery)
เพิ่มรูปภาพ (Add Photo)

ดาวน์โหลด APK

ดาวน์โหลดแอปพลิเคชันเพื่อใช้งานบนอุปกรณ์ Android

กรุณาดาวน์โหลดไฟล์ APK ด้านล่างเพื่อติดตั้งแอปพลิเคชันบนอุปกรณ์ Android ของคุณ
